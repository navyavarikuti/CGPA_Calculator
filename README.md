# CGPA Calculator

A simple and responsive CGPA Calculator built using HTML, CSS, and JavaScript.

## Features

* Enter the number of semesters (up to 8)
* Dynamic SGPA input field generation
* Input validation for user inputs
* Instant CGPA calculation
* Clean and user-friendly interface

## Screenshots

### 1. Entering Number of Semesters

Displays the input field where users can enter the number of semesters.

### 2. Dynamic SGPA Fields

Generates SGPA input fields based on the number of semesters entered.

### 3. CGPA Calculation

Calculates and displays the CGPA after entering SGPA values.

### 4. Validation Alert

Shows an alert message when invalid or incomplete data is entered.
